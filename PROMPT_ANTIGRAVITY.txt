obrigacoes fiscais reais para dezenas de postos de combustivel, 24h/dia.

## COMO SUBIR/DERRUBAR O SISTEMA (unico jeito aprovado)

- Subir:    duplo clique / cmd /c "C:\SpedGenerator\iniciar_sistema.bat"
- Derrubar: cmd /c "C:\SpedGenerator\finalizar_sistema.bat"
- Pausar/retomar SEM derrubar: via monitor web (8777) ou
  controle.json ("pausado"/"normal") — prefira o monitor.
- Python: "C:\Users\SERVIDOR SPED\AppData\Local\Python\pythoncore-3.14-64\python.exe"
- Postgres local: localhost:5432, user postgres, senha 123.
- O ACS Gerente SO abre bancos *_local no localhost (o acsgerente.ini e
  reescrito pelo ini_manager a cada geracao com Caminho={base}_local).
  O banco do cliente no servidor NUNCA e aberto pelo ACS.
